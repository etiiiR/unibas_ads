\begin{onexercise}
  For this (and later) exercise sheets you need Java. Please install it on your
  system. For Ubuntu this is for example possible with \texttt{sudo apt install
  openjdk-21-jdk}. For other operating systems you can obtain it
  at \url{https://www.azul.com/downloads/?version=java-21-lts&package=jdk#zulu}.
\medskip

\emph{You can form teams of arbitrary size to hand in the exercises. However, 
please make sure that your entire team is in the same exercise group, and
that you stay within the same team during the semester. We recommend teams of 
two.}

\medskip

\emph{When preparing your handin, please follow these steps:
\begin{enumerate}[\hspace{5mm}1.]
  \itemsep0em
  \item Delete the files generated by gradle with \texttt{./gradlew clean} or 
    \texttt{gradlew.bat clean} respectively.
  \item Add your solutions of exercise 1.1 as a PDF or text file.
  \item Create a zip archive of the entire folder.
  \item Upload your archive on ADAM.
\end{enumerate}
If you wish to upload a new version of your handin, please make sure that it 
has the same file name and is uploaded from the same account.}

\bigskip
\end{onexercise}
